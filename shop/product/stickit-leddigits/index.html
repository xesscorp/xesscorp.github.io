StickIt!-LedDigits

  • Eight 7-segment LEDs
  • Needs only 8 I/O pins!
  • PMOD compatible
  • Wing compatible
  • Open-source design

This product is currently unavailable.

  • StickIt! LED Digits module manual (PDF file)

Currently unrated


Well, you can probably tell what this module does. Just connect it to a PMOD or Wing socket on the StickIt! board and your XuLA can display up to eight digits of numerical goodness! And, through the magic of Charlieplexing, you can individually control all 56 LED segments using only eight I/O pins of the FPGA!

As with the StickIt! board, we made the entire StickIt! LED Digits module design open-source. All of it: bitstreams, schematic, PCB layout, documentation — everything!

FAQ

I ordered a StickIt! LED Digits module and all I got was a board. No software? No manual? No fancy box!?

When you order a StickIt! LED Digits module, all you get is the board. That's it. The StickIt! LED Digits module doesn't need any software itself. And the manual is online. As for the fancy box, well, sorry about that. Think of it as "saving the environment".

What kind of instructions are there for using the StickIt! LED Digits module ?

The StickIt! LED Digits module manual will tell you how to install your board, and it shows how the LEDs are connected to the I/O pins. And there is a design example online that shows how to use it.
